# Sales & Shipment Performance Dashboard (Power BI)

## Project Overview
This project is an interactive Power BI dashboard built to analyze sales, shipment volume, profitability, and salesperson performance across products and regions.

The dashboard helps identify trends, top-performing products, loss-making items, and regional sales distribution using business KPIs and time-based analysis.

---

## Tools & Technologies
- Power BI
- DAX
- Data Modeling (Star Schema)
- Power Query
- Data Visualization

---

## Data Model
- Fact Table: Shipments
- Dimension Tables:
  - Calendar
  - Products
  - People (Salesperson)
  - Locations

A star schema model was implemented to support efficient filtering and time intelligence.

---

## Key Measures (DAX)
- Total Amount
- Total Boxes
- Total Profit
- Profit %
- Shipment Count
- CY vs PY comparisons

---

## Dashboard Highlights
- KPI cards for revenue, profit, shipment volume, and margin
- Year-over-Year (CY vs PY) sales and shipment trends
- Product-wise revenue and profit analysis
- Salesperson performance ranking
- Geographic sales distribution
- Shipment size distribution analysis

---

## Key Insights
- Sales and shipment volumes show a declining trend compared to the previous year
- A small set of products contributes the majority of revenue
- Some high-selling products are loss-making, indicating pricing or cost issues
- Top salespersons consistently maintain strong profit margins
